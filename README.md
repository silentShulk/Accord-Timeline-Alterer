# Accord's Timeline Alterer (ATA)

This is a mod manager for NieR:Automata for Linux created using the Rust programming language

It is an expansion of another repo that you can find here:
```
https://github.com/silentShulk/Nier-Automata-Modding-Linux.git
```



## Features of ATA
ATA allows you to :
- Install mods (you can find mods on Nexus Mods)
- Uninstall mods
- Disable a mod without uninstalling it
- List mods you have installed
